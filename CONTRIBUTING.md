# Contributing

Thank you for improving Plant Hunter.

1. Create a focused branch from `main`.
2. Keep credentials and production data out of source, fixtures, screenshots, and logs.
3. Add or update tests for behavior and security changes.
4. From `frontend/`, run `npm test`, `npm run build`, and `npm audit`.
5. Explain user-visible behavior, data-handling changes, and migration needs in the pull request.

Only contribute code, copy, fonts, images, audio, video, and other assets that you are authorized to redistribute under terms compatible with this repository. Preserve existing attribution and copyright notices.

Report vulnerabilities privately according to [`SECURITY.md`](SECURITY.md), not through a public issue.
